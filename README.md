# 🔐 Secure Exam Evaluation System

## 📌 Overview

This project is a secure and intelligent online exam evaluation system that protects student data during processing using Confidential Computing concepts.

The system simulates a Trusted Execution Environment (TEE), where answers are encrypted before evaluation and processed securely without exposing sensitive data.

---

## 🚀 Features

* 🔐 Data Encryption before processing
* 🛡 Secure Evaluation (TEE Simulation)
* 🤖 AI-based Answer Scoring
* 📊 Score Visualization (Progress Bar)
* 📜 Evaluation Logs for transparency
* 🎨 Modern and user-friendly interface

---

## ⚙️ Tech Stack

* Backend: Python (Flask)
* Frontend: HTML, CSS, JavaScript
* AI Logic: Keyword-based intelligent scoring
* Security: Base64 Encryption (simulation)

---

## 🧠 How It Works

1. User enters answer
2. Answer is encrypted
3. Data is processed in secure function (TEE simulation)
4. AI evaluates the answer
5. Score and feedback are displayed

---

## 🎯 Innovation

Unlike traditional systems, this project ensures **data-in-use security**, meaning data is protected not only when stored or transmitted, but also during processing.
